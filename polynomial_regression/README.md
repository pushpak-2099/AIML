# Polynomial Regression

## Overview
This project demonstrates Polynomial Regression, an extension of Linear Regression used to model non-linear relationships between variables.

The model transforms input features into higher-degree polynomial features to better fit curved data patterns.

---

## Objective
- Understand limitations of Linear Regression on non-linear data
- Learn polynomial feature transformation
- Visualize curve fitting
- Evaluate model performance

---

## How It Works
1. Load dataset
2. Apply polynomial feature transformation
3. Train Linear Regression model
4. Visualize results
5. Evaluate performance
